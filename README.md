### Prerequisites

- Node.js (v18 or higher)

### Installation

1. Clone the repository

```bash
git clone https://github.com/Suresh-cs-q/portfolio.git
```

2. Navigate to project directory

```bash
cd portfolio
```

3. Install dependencies

```bash
npm install
```

4. Start development server

```bash
npm run dev
```

5. Build for production

```bash
npm run build
```
